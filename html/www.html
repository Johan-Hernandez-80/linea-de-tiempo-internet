Contexto histórico

Tim Berners-Lee, ingeniero en el CERN, propuso en 1989 un sistema de hipertexto para compartir documentos científicos entre investigadores distribuidos. Su idea era simple: enlazar documentos mediante referencias que el usuario pudiera seguir con un clic.

Tecnologías fundamentales

La Web se soporta en tres componentes clave: URLs para localizar recursos, HTTP como protocolo de transferencia, y HTML como lenguaje de marcado para estructurar documentos. El primer servidor y navegador se implementaron en una estación NeXT alrededor de 1990.

Difusión y efectos

A partir de 1993, con la aparición de navegadores gráficos como Mosaic, la Web experimentó un crecimiento exponencial. Facilitó la publicación de contenidos, el comercio electrónico y cambió cómo accedemos a la información en la vida cotidiana.

Desarrollo técnico

  • HTML evolucionó desde simples etiquetas hasta versiones con CSS y JavaScript que habilitan experiencias interactivas.
  • HTTP ha evolucionado (HTTP/1.1 → HTTP/2 → HTTP/3) para mejorar rendimiento y seguridad.

Impacto cultural y económico

La Web democratizó la publicación y acceso a la información y habilitó nuevas industrias: buscadores, redes sociales, comercio electrónico y servicios en la nube.
