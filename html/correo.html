Contexto histórico

El correo electrónico surgió dentro de ARPANET como una herramienta para que los investigadores se comunicaran rápidamente. En 1971 Ray Tomlinson, ingeniero de BBN, envió el primer mensaje entre dos hosts conectados por ARPANET.

Detalles del primer envío

El mensaje enviado fue una prueba sin contenido significativo, pero marcó un hito en la historia de Internet. Tomlinson eligió el carácter @ para separar el nombre de usuario del host, decisión que estableció el formato estándar de las direcciones de correo.

Funcionamiento básico

El correo electrónico se basa en protocolos como SMTP (Simple Mail Transfer Protocol) para enviar mensajes y POP/IMAP para recibirlos. Mensajes, cabeceras y adjuntos se estructuran para permitir interoperabilidad entre sistemas distintos.

Impacto social y técnico

El correo transformó la comunicación profesional y personal, permitiendo correspondencia asíncrona y archivo de mensajes. Fue la primera aplicación de gran adopción en Internet y sigue siendo clave en comunicaciones formales y de negocio.

Problemas y evolución

  • Spam y filtrado: la adopción masiva trajo la necesidad de filtros y listas negras.
  • Seguridad: con el tiempo se añadieron protocolos como TLS para cifrar conexiones y PGP para cifrar mensajes.
  • Escalabilidad: servicios comerciales y proveedores masivos como Gmail y Outlook cambiaron la forma de gestionar el correo a gran escala.
