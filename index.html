Línea de tiempo del internet

1969: ARPANET

Primera red de computadoras que sentó las bases de lo que hoy conocemos como Internet.

1971: Primer correo electrónico

Se envía el primer email, introduciendo el uso del símbolo @ para direcciones.

1983: Protocolo TCP/IP

Se establece el estándar global de comunicación entre redes.

1989: Nace la World Wide Web

Se propone la Web como sistema para compartir información mediante hipertexto.

1998: Fundación de Google

Revoluciona la forma de encontrar información en Internet.

2004: Redes sociales masivas

Las plataformas sociales transforman la interacción digital global.

2007: Internet móvil moderno

Los smartphones llevan Internet al uso cotidiano y portátil.

2009: Blockchain y Bitcoin

Surge una nueva arquitectura descentralizada para transacciones digitales.

2019: Despliegue inicial de 5G

Mayor velocidad y menor latencia en conectividad global.

2022: IA generativa

Herramientas capaces de crear texto, imágenes y código se popularizan globalmente.
